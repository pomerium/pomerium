# pomerium/pomerium/examples — moved

The runnable Pomerium examples that used to live in this directory have moved
to the documentation repository:

> **https://github.com/pomerium/documentation/tree/main/content/examples**

The documentation site already hosted parallel copies of most of these
examples and has been the de-facto source of truth for some time. Maintaining
a second copy here produced silent drift; this directory has been retired so
the docs site is the single source of truth.

If you arrived here from a tutorial or third-party link expecting source
files, the equivalent content is at the URL above. The actively-maintained
mTLS reference now lives at
[`content/examples/mutual-tls`](https://github.com/pomerium/documentation/tree/main/content/examples/mutual-tls).
